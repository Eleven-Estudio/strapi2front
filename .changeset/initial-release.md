Initial release of strapi2front - Generate TypeScript types, services, and framework actions from your Strapi schema.

Features:
- Auto-generate TypeScript types from Strapi schema
- Create typed services for collections and single types
- Generate Astro Actions for client/server data fetching
- Automatic Strapi version detection (v4/v5)
- Smart project detection (framework, TypeScript, package manager)
- Multiple output structures (by-feature, flat)
