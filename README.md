# strapi-integrate

> CLI for seamlessly integrating Strapi CMS with modern frontend frameworks

[![npm version](https://img.shields.io/npm/v/strapi-integrate.svg)](https://www.npmjs.com/package/strapi-integrate)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Features

- **Type Generation** - Auto-generate TypeScript types from your Strapi schema
- **Service Generation** - Create typed services for all your content types
- **Astro Actions** - Generate type-safe Astro Actions for client/server data fetching
- **Smart Detection** - Automatically detects your project setup (framework, TypeScript, package manager)
- **Schema Sync** - Keep your types in sync with Strapi schema changes

## Quick Start

```bash
# Initialize in your Astro project
npx strapi-integrate init

# Sync when your Strapi schema changes
npx strapi-integrate sync
```

## Requirements

- Node.js 18+
- Strapi v5
- Astro 4+ (more frameworks coming soon)
